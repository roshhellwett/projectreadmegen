How to Use projectreadmegen

Follow these steps to generate your README

1

Get Your Folder Structure

Run this command in your project folder to get the tree structure:

tree /F /A
💡 On Mac use find . -print | sed 's|[^/]*/| |g' or install tree via Homebrew
2

Paste the Tree Structure

Copy the output and paste it into the "Folder Tree" text area on the left panel.

Paste here
3

Configure Options

Fill in optional details to customize your README:

  • 👤 Author Name - Your name
  • 🐙 GitHub Username - For badges and links
  • 📄 Template - Choose from Minimal, Standard, Full, or Academic
4

Generate & Download

Click "Generate README" and get your ready-to-use README!
